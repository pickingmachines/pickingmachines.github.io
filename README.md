# Picking Machines

Mobile manipulation robotics for agriculture.

**Live site:** [pickingmachines.github.io](https://pickingmachines.github.io/)

## Repository layout

This is a [GitHub Pages](https://docs.github.com/en/pages/getting-started-with-github-pages/about-github-pages) user site (`username.github.io`). The published site is built from the **default branch** at the **repository root**:

- `index.html` — main page (entry point for the site)
- `logo.png`, `bg.jpg` — static assets referenced by the page

`README.md` is for people browsing the repo on GitHub; it is **not** served as the website homepage. GitHub Pages uses `index.html` (or `index.md` with Jekyll) at the root.

## Local preview

Open `index.html` in a browser, or from this directory run a static server, for example:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
